### Step 2: Deploy to GitHub (10 minutes)

#### A. Open Command Prompt/Terminal
- **Windows**: Press `Win + R`, type `cmd`, press Enter
- **Mac**: Press `Cmd + Space`, type `terminal`, press Enter

#### B. Run These Commands
Copy and paste each line, press Enter after each:

```bash
cd C:\Users\ADMIN\OneDrive\Desktop\rawan-resume

git init

git add .

git commit -m "Initial commit: Professional resume website"

git remote add origin https://github.com/Rawan-J-Othman/resume.git

git branch -M main

git push -u origin main
```

**Note**: You'll be asked for your GitHub username and password/token.

---

### Step 3: Enable GitHub Pages (2 minutes)

1. Go to: **https://github.com/Rawan-J-Othman/resume**
2. Click **"Settings"** (top menu)
3. Click **"Pages"** (left sidebar)
4. Under "Source":
   - Branch: Select **"main"**
   - Folder: Select **"/ (root)"**
5. Click **"Save"**
6. Wait 2-3 minutes
7. Visit: **https://rawan-j-othman.github.io/resume/**
