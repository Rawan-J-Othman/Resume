## Deploy to GitHub Pages

### Option 1: Create New Repository

1. Go to [GitHub](https://github.com) and create a new repository
2. Name it something like `nutrition-resume` or `wellness-portfolio`
3. Initialize the repository (you can skip adding README since we have one)
4. Clone the repository to your computer:
   ```bash
   git clone https://github.com/YOUR-USERNAME/YOUR-REPO-NAME.git
   ```
5. Copy all files from this folder into the cloned repository folder
6. Commit and push:
   ```bash
   git add .
   git commit -m "Initial commit: Nutrition resume website"
   git push origin main
   ```
7. Go to your repository Settings → Pages
8. Under "Source", select "main" branch
9. Click Save
10. Your site will be live at: `https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/`

### Option 2: Use GitHub Desktop (Easier)

1. Download and install [GitHub Desktop](https://desktop.github.com/)
2. Sign in with your GitHub account
3. Click "Create New Repository"
4. Name it and choose this folder as the location
5. Click "Publish repository"
6. Go to GitHub.com → Your repository → Settings → Pages
7. Enable GitHub Pages from the main branch
8. Your site will be live!
